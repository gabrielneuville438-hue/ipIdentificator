# IpIdentificator

## 🇧🇷 Português

### Descrição
O IpIdentificator é uma ferramenta desenvolvida em Java para identificar automaticamente a sub-rede local, realizar a varredura de endereços IP, detectar portas comuns abertas e gerar um relatório de segurança.

### Funcionalidades
- Identificação automática da sub-rede.
- Varredura de hosts ativos.
- Verificação de portas comuns.
- Geração automática de relatório (.txt).
- Uso de multithreading para acelerar a análise.

### Como executar

Compile o programa:

```bash
javac ipIdentificator.java
```

Execute:

```bash
java ipIdentificator
```

O programa detectará automaticamente a sub-rede, realizará a varredura e gerará o arquivo `relatorio_seguranca.txt`.

## 🇺🇸 English

### Description

IpIdentificator is a Java application that automatically detects the local subnet, scans IP addresses, checks common open ports, and generates a security report.

### Features
- Automatic subnet detection.
- Active host scanning.
- Common port detection.
- Automatic report generation (.txt).
- Multithreaded scanning for better performance.

### How to run

Compile:

```bash
javac ipIdentificator.java
```

Run:

```bash
java ipIdentificator
```

The application automatically detects the local subnet, performs the scan, and generates the `relatorio_seguranca.txt` report.
