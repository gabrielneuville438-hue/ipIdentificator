# 🔐 Secure Password Generator (CSPRNG)

<details>
<summary><b>🇧🇷 Clique aqui para ler em Português</b></summary>
<br>

Gerador de senhas aleatórias criptograficamente seguro desenvolvido em **Java** para fins de cibersegurança e proteção de credenciais.

## 🚀 Funcionalidades
* **Alta Entropia:** Utiliza `java.security.SecureRandom` em vez de geradores pseudoaleatórios comuns.
* **Garantia de Complexidade:** Força a inclusão de pelo menos 1 letra maiúscula, 1 minúscula, 1 número e 1 caractere especial.
* **Automação Prática:** Envia a senha gerada diretamente para a Área de Transferência (**Clipboard**) do sistema operacional de forma invisível.
* **Sem Dependências:** Desenvolvido em Java nativo puro, sem necessidade de pacotes externos.

## 💻 Como Compilar e Executar
Abra o terminal no diretório do projeto e execute os comandos:
```bash
javac PasswordGenerator.java
java PasswordGenerator
```
Após a conclusão, basta colar (**Ctrl + V**) a senha onde desejar!

A cryptographically secure random password generator developed in **Java** for cybersecurity purposes and credential protection.

## 🚀 Features
* **High Entropy:** Uses `java.security.SecureRandom` instead of weak pseudo-random generators.
* **Complexity Guaranteed:** Forces the inclusion of at least 1 uppercase letter, 1 lowercase letter, 1 number, and 1 special character.
* **Seamless Automation:** Copies the generated password directly to the operating system's **Clipboard** automatically.
* **Zero Dependencies:** Built entirely with pure native Java, requiring no external packages.

## 💻 How to Compile and Run
Open your terminal in the project directory and execute:
```bash
javac PasswordGenerator.java
java PasswordGenerator
```
Once done, simply paste (**Ctrl + V**) the password wherever you need it!
